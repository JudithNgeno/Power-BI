# POWER-BI
## OVERVIEW
This repository documents a Power BI dashboard that analyses UK traffic camera activity between 2021 and 2023 using open data from the Office for National Statistics (ONS) "Traffic camera activity" dataset. The project explores how movement patterns for pedestrians, cyclists, cars, buses, vans, and trucks evolved acoss the Greater Manchester region in the period following the COVID-19 pandemic.
